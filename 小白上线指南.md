## 第1步：注册 Supabase（免费数据库）

Supabase 就是我们的"云端数据库"，免费，不用自己搭服务器。

### 1.1 打开注册页面

1. 用浏览器打开：https://supabase.com
2. 点击右上角 **"Start your project"** 按钮
3. 选择用 **GitHub账号** 登录（推荐，最方便）
   - 如果没有GitHub账号，点 "Sign up" 用邮箱注册一个
4. 登录后进入 Supabase 控制台

### 1.2 创建项目

1. 点击 **"New Project"** 大按钮
2. 填写信息：
   - **Name**：输入 `lzdzhixuan`（随便起，这个只是项目名）
   - **Database Password**：设置一个密码（**一定要记住！写下来！**）
   - **Region**：选择 **Northeast Asia (Tokyo)**（离中国最近）
3. 点击 **"Create new project"**
4. 等待约2分钟，出现 Dashboard 界面就说明创建成功了

### 1.3 拿到你的密钥

1. 在左侧菜单点击 **齿轮图标（Settings）**
2. 点击 **API**
3. 你会看到两个重要信息：
   - **Project URL**：像 `https://abcdefghijk.supabase.co` 这样的一串地址
   - **anon public**：像 `eyJhbGciOiJIUzI1NiIs...` 这样的一串长字符

4. **把这两个信息复制保存下来**（后面要用）

> ⚠️ 这两个信息是公开的，放在前端代码里是安全的，不用担心泄露。

## 第2步：建数据库表

### 2.1 打开SQL编辑器

1. 在 Supabase 左侧菜单点击 **"SQL Editor"**（图标像一个闪电）
2. 点击 **"New query"** 按钮

### 2.2 粘贴建表代码

把下面的所有代码，**全部复制**，粘贴到编辑器里：

```sql
-- 先建供应商表（商品表依赖它）
CREATE TABLE suppliers (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  company_name TEXT NOT NULL,
  contact_person TEXT,
  phone TEXT,
  main_categories JSONB DEFAULT '[]',
  brand TEXT,
  annual_capacity TEXT,
  rating DECIMAL(2,1) DEFAULT 0,
  level TEXT CHECK (level IN ('A', 'B', 'C')),
  status TEXT DEFAULT '活跃',
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 商品表
CREATE TABLE products (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  name TEXT NOT NULL,
  images JSONB DEFAULT '[]',
  color_season TEXT NOT NULL,
  style_type TEXT NOT NULL,
  stock INTEGER DEFAULT 0,
  wholesale_price DECIMAL(10,2) NOT NULL,
  retail_price DECIMAL(10,2) NOT NULL,
  attributes JSONB DEFAULT '{}',
  description TEXT,
  supplier_id UUID REFERENCES suppliers(id),
  is_hot BOOLEAN DEFAULT false,
  status TEXT DEFAULT 'active',
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 订单表
CREATE TABLE orders (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  items JSONB DEFAULT '[]',
  total_amount DECIMAL(10,2) NOT NULL,
  status TEXT DEFAULT '待支付',
  shipping_name TEXT,
  shipping_phone TEXT,
  shipping_address TEXT,
  payment_method TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- VIP会员表
CREATE TABLE vip_members (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  name TEXT,
  phone TEXT,
  level TEXT DEFAULT 'V1' CHECK (level IN ('V1', 'V2', 'V3')),
  annual_spend DECIMAL(12,2) DEFAULT 0,
  discount_rate DECIMAL(3,2) DEFAULT 1.00,
  return_rate INTEGER DEFAULT 7,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 课程表
CREATE TABLE courses (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  title TEXT NOT NULL,
  price DECIMAL(10,2) NOT NULL,
  level TEXT NOT NULL,
  description TEXT,
  outline JSONB DEFAULT '[]',
  created_at TIMESTAMPTZ DEFAULT now()
);
```

3. 点击右下角绿色的 **"Run"** 按钮
4. 看到 "Success" 提示就说明建表成功了！

### 2.3 设置数据安全策略

再点 **"New query"**，粘贴下面的代码并运行：

```sql
-- 开启安全策略
ALTER TABLE products ENABLE ROW LEVEL SECURITY;
ALTER TABLE suppliers ENABLE ROW LEVEL SECURITY;
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;
ALTER TABLE vip_members ENABLE ROW LEVEL SECURITY;
ALTER TABLE courses ENABLE ROW LEVEL SECURITY;

-- 商品、供应商、课程：所有人可看
CREATE POLICY "公开读取商品" ON products FOR SELECT USING (true);
CREATE POLICY "公开读取供应商" ON suppliers FOR SELECT USING (true);
CREATE POLICY "公开读取课程" ON courses FOR SELECT USING (true);

-- 订单和VIP：只能看自己的
CREATE POLICY "用户读取自己的订单" ON orders FOR SELECT
  USING (auth.uid() = user_id);
CREATE POLICY "用户读取自己的VIP" ON vip_members FOR SELECT
  USING (auth.uid() = user_id);
```

点 **Run**，看到 Success 就完成了！

### 2.4 创建图片存储桶

1. 左侧菜单点击 **"Storage"**
2. 点击 **"New Bucket"**
3. 名称输入：`product-images`
4. 勾选 **"Public bucket"**
5. 点击 **"Create bucket"**

✅ **Supabase 配置完成！**

## 第4步：部署到 Vercel（免费，5分钟上线）

Vercel 是一个免费的网站托管平台，专门用来部署 Next.js 网站。

### 4.1 注册 Vercel

1. 打开 https://vercel.com
2. 点击 **"Sign Up"**
3. 选择 **"Continue with GitHub"**（用刚才的GitHub账号登录）
4. 授权 Vercel 访问你的 GitHub

### 4.2 导入项目

1. 登录后，点击 **"Add New..."** → **"Project"**
2. 在列表中找到你刚才创建的 `lzdzhixuan` 仓库
3. 点击 **"Import"**

### 4.3 配置环境变量（重要！）

1. 在部署页面，找到 **"Environment Variables"** 区域
2. 添加两条：
   - Name: `NEXT_PUBLIC_SUPABASE_URL`  Value: `你的Supabase URL`
   - Name: `NEXT_PUBLIC_SUPABASE_ANON_KEY`  Value: `你的Supabase Key`
3. 点击 **"Deploy"** 按钮

### 4.4 等待部署完成

约2-3分钟后，你会看到 🎉 庆祝动画！

你的网站地址类似：`lzdzhixuan.vercel.app`

**点这个地址，你的网站就上线了！**

## 第5步：绑定你的域名

你说你在腾讯云买了域名，现在把它绑定到 Vercel。

### 5.1 在 Vercel 添加域名

1. 进入你的项目 → **Settings** → **Domains**
2. 输入你的域名，比如 `www.lzdzhixuan.com`
3. 点击 **"Add"**
4. Vercel 会显示需要添加的 DNS 记录，通常是：
   - 类型: `CNAME`
   - 名称: `www`
   - 值: `cname.vercel-dns.com`

### 5.2 在腾讯云配置 DNS

1. 登录腾讯云控制台 → **域名注册** → 找到你的域名
2. 点击 **"解析"**
3. 添加记录：
   - **记录类型**: CNAME
   - **主机记录**: www
   - **记录值**: cname.vercel-dns.com
4. 再添加一条（让不带www也能访问）：
   - **记录类型**: CNAME
   - **主机记录**: @
   - **记录值**: cname.vercel-dns.com

5. 等待5-10分钟，DNS生效

### 5.3 SSL证书

Vercel 自动帮你配好 HTTPS，不用额外操作！

✅ **完成！打开你的域名就能看到网站了！**

## 常见问题

### Q: 网站打开是空白页？
检查 `.env.local` 或 Vercel 环境变量里的 Supabase URL 和 Key 是否正确。

### Q: 域名打不开？
DNS 需要几分钟生效，最长等30分钟。

### Q: 想修改网站内容？
改代码 → 推送到 GitHub → Vercel 自动重新部署（约1分钟）

### Q: 想添加商品数据？
在 Supabase → Table Editor → products 表 → Insert Row 手动添加

### Q: 小程序怎么做？
最简单方式：用微信开发者工具创建项目，用 `<web-view src="https://你的域名">` 嵌入网站即可
