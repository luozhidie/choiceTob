## 二、上线路线图

### Phase 1：内容填充（Day 1-3）

| 天数 | 目标 | 具体任务 |
|------|------|---------|
| Day 1 | 品牌视觉就绪 | 上传首页5张图片 + 联系信息企业微信码 + 微信/支付宝收款二维码 |
| Day 2 | 核心业务数据 | 录入5件买手选品(含供货价) + 3组每日搭配 |
| Day 3 | 丰富内容 | 3个陈列案例 + 2门课程 + 2个设计作品 |

### Phase 2：全链路测试（Day 4-5）

| 测试项 | 操作 | 通过标准 |
|--------|------|---------|
| VIP购买→确认→生效 | 注册测试账号→购买→后台确认 | 权益即时生效 |
| 搭配付费墙 | 非VIP看模糊+VIP免费 | 权限区分正确 |
| 设计付费墙 | 非VIP/基础VIP付费+高阶VIP免费 | 权限区分正确 |
| 企划提交 | 选类型→填信息→提交 | 订单创建成功 |
| 风格测试 | VIP免费/非VIP¥99 | 标签切换正确 |

### Phase 3：种子客户（Day 6-14）

```
目标：5个免费体验客户
动作：私聊5个熟悉的服装店主 → 后台开通VIP → 引导使用 → 收集反馈
```

### Phase 4：正式推广（Day 15-30）

```
目标：首月3个付费客户
渠道：朋友圈 + 微信群 + 小红书 + 抖音
营收预期：基础VIP×3 = ¥29,400
```

### Phase 5：规模化（Month 2-3）

```
目标：10个付费客户 + 1个企划订单
月营收预期：¥15,000~18,000/月
```

## 五、风险与应对

| 风险 | 概率 | 应对 |
|------|------|------|
| 内容不足导致用户流失 | 高 | Day1-3务必填满核心内容，至少各3条 |
| 收款码未配置影响付费 | 高 | Day1优先上传收款二维码 |
| ¥9,800定价吓跑客户 | 中 | 提供免费体验期（后台手动开通1个月） |
| 竞品低价竞争 | 低 | 强调中高端定位+专业服务+供货价独占性 |
