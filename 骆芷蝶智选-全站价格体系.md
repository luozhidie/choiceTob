## 五、定价策略总结

### 3层价格漏斗

```
引流层（¥0~¥199）→ 核心层（¥9,800~¥29,800/年）→ 高价值层（¥8,000~¥49,800/次）

风格测试¥99 → 企划工具¥99 → 杂志¥138/月
                                    ↓ 转化
                            基础VIP ¥9,800/年
                                    ↓ 升级
                            高阶VIP ¥29,800/年
                                    ↓ 深度
                        色彩/风格企划 ¥8,000
                        标准企划 ¥12,000
                        全案企划 ¥49,800
```

### 核心定价原则

| 原则 | 体现 |
|------|------|
| **单买不如买VIP** | 搭配月费¥999 > VIP月均价¥817 |
| **VIP逐级递增** | 基础¥9,800 → 高阶¥29,800，3倍差价对应3倍权益 |
| **企划梯度清晰** | 轻量¥8,000 → 标准¥12,000 → 全案¥49,800 |
| **新客破冰** | 全案新客¥19,800（4折），降低首单门槛 |
| **中高端定位** | 最低年费¥9,800起，筛选有付费能力的门店 |

### 客单价与续费策略

| 场景 | 策略 |
|------|------|
| 新客户首单 | 全案企划新客¥19,800 或 基础VIP免费体验1个月 |
| 基础→高阶升级 | 线下补差价，按剩余天数折算 |
| 年度续费 | 提前30天续费享9折（后台手动调整到期时间） |
| 多店客户 | 多店可谈团购价（如3店基础VIP ¥25,000） |
